object-fit →background-sizeと似た役割

object-position→background－positionと似た役割

このobject-　君img タグでなく動画つまりvideo タグにも対応している優れもの！
object-fit:; の値は
fill　　初期値
contain　
cover
none
scale-down


object-position の値は
ちなみに
こいつを頑張れば
画像が動画みたいに出来る～～
html
<img src="img/haikyo.webp" alt=""width="500" height="500">
CSS
body{
  padding:100px;
  padding-left:200px;
}
img {
 
  object-fit:cover;   
  object-position: 0% 0%;
  animation: ltr 5s alternate infinite linear;
  }
  /* alternateは逆再生infiniteは回数無限linearは動きを同じで */
  @keyframes ltr {
  0% {
  object-position: 0% 0%;
  }
  50% {
  object-position: 50% 0%;
  }
  100% {
  object-position: 100% 0%;
  }
  }
  /* 、object-position: 0% 0%は左上になり、object-position: 100% 100%は右下になります。 */
  /* 今回はX軸のみ動かしました */
